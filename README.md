# ATSUKAN-V2

ATSUKAN-V2 is a type of GPU burner which searches most efficient B/f ratio.

## Installation
```bash
git clone https://github.com/enp1s0/atsukan-v2
cd atsukan-v2
make
```

## Usage
### Single GPU mode
```
./atsukan.out
```

### Multi GPU mode
```
mpirun -np [num_gpus] ./atsukan.out
```
